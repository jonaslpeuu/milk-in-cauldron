Follow these steps to install Milk in Cauldron on your Minecraft

Prerequisites

Before installing the mod, make sure you have:

  • Minecraft Java Edition (check supported versions)
  • Fabric Loader (latest version recommended)
  • Fabric API (matching your Minecraft version)

Step 1: Install Fabric Loader

If you haven't already installed Fabric Loader, follow these steps:

  1. Go to Fabric's official website
  2. Download the Fabric Installer
  3. Run the installer and select your desired Minecraft version
  4. Click "Install" and wait for completion
  5. Launch Minecraft and select the "fabric-loader-1.21.10" profile

Step 2: Download Fabric API

Fabric API is required for most Fabric mods, including Milk in Cauldron:

  1. Go to Fabric API on Modrinth
  2. Select the version matching your Minecraft version
  3. Download the JAR file

Step 3: Download Milk in Cauldron

Get the mod from one of these sources:

  • Modrinth (recommended)
  • GitHub Releases

Download the JAR file for your Minecraft version

Step 4: Install the Mods

Place both mod files in your Minecraft mods folder:

Windows

%appdata%\.minecraft\mods

macOS

~/Library/Application Support/minecraft/mods

Linux

~/.minecraft/mods

Your mods folder should contain:

  • fabric-api-[version].jar
  • milk_in_cauldron-[version].jar

Step 5: Launch the Game

  1. Open the Minecraft Launcher
  2. Select the fabric-loader-1.21.10 profile
  3. Click Play
  4. Wait for the game to load
  5. Check the mod list (Mods button on title screen) to verify installation

Troubleshooting

Mod not showing in mod list?

Make sure you're using the Fabric profile and both JAR files are in the mods folder.

Game crashes on startup?

Check that your Fabric Loader and Fabric API versions are compatible with Minecraft 1.21.10.

Texture is missing (pink/black squares)?

Try pressing F3+T to reload textures, or reinstall the mod.
