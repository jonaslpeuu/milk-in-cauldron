General Questions

What does this mod do?

Milk in Cauldron is a simple quality-of-life mod that adds the ability to pour milk from a bucket into a cauldron, just like you can with water, lava, or powder snow in vanilla Minecraft.

What Minecraft version is supported?

Currently, the mod supports Minecraft 1.21.10 with Fabric Loader.

Is this mod available for Forge?

No, this mod is currently only available for Fabric. There are no plans for a Forge version at this time.

Can I use this mod in modpacks?

Yes! You're welcome to include this mod in your modpack. Just please credit the author.

Compatibility

Does this mod work with other cauldron mods?

It should work with most other mods, but there may be conflicts with mods that heavily modify cauldron behavior. If you encounter issues, please report them on GitHub.

Does this work on servers?

Yes! The mod works on both client and server. For multiplayer, the mod needs to be installed on both the server and all clients.

Does this work with OptiFine/Sodium?

Yes, the mod is compatible with performance mods like Sodium, Lithium, and OptiFine alternatives.

Features & Gameplay

Does drinking from the milk cauldron cure potion effects?

No, the milk in the cauldron is purely decorative. You cannot drink from it directly. If you need to cure potion effects, retrieve the milk with a bucket and drink it normally.

Can I partially fill the cauldron?

No, currently the cauldron fills completely with one bucket. Partial filling may be added in a future update.

Will rain affect the milk cauldron?

No, rain does not affect milk cauldrons. Unlike empty cauldrons that fill with water during rain, milk cauldrons will keep their milk.

Can I dye items in milk like in water?

No, the milk cauldron doesn't support dyeing leather armor or other items. It's purely for milk storage.

Troubleshooting

The milk texture shows pink/black squares!

This indicates a missing texture. Try these solutions:

  1. Press F3 + T to reload textures
  2. Reinstall the mod
  3. Make sure you're using the latest version

The game crashes on startup!

Check the following:

  • Ensure you have Fabric API installed
  • Verify your Fabric Loader version is 0.18.2 or higher
  • Check for mod conflicts by testing with only this mod installed

Right-clicking the cauldron does nothing!

Make sure you're:

  • Using the correct item (Milk Bucket to fill, Empty Bucket to empty)
  • Clicking on the correct cauldron (empty for filling, full milk cauldron for emptying)
  • The mod is properly installed (check mod list)
