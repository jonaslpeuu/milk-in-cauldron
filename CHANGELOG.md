# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-12-14

### Added
- Initial release
- Milk bucket can now be used on empty cauldrons to fill them with milk
- Empty bucket can be used on full milk cauldrons to retrieve milk
- Custom milk cauldron block with white milk texture
- Sound effects for filling and emptying
- English and German translations

### Technical
- Compatible with Minecraft 1.21.10
- Requires Fabric Loader 0.18.2+
- Requires Fabric API
